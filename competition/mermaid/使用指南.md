## 方法一：在线编辑器（推荐）

### Mermaid Live Editor

1. 访问：https://mermaid.live/
2. 打开对应的 `.mmd` 文件，复制全部内容
3. 粘贴到左侧编辑器
4. 右侧实时预览效果
5. 点击右上角 **Download PNG** 或 **Download SVG**

### 设置导出参数

```
推荐设置：
- 分辨率: 300 DPI（适合打印）
- 格式: PNG（Word兼容）或 SVG（可缩放）
- 背景: 白色
- 尺寸: 自动
```

## 方法二：draw.io

1. 访问：https://app.diagrams.net/
2. 创建新图表
3. 菜单：**Arrange** → **Insert** → **Advanced** → **Mermaid**
4. 粘贴代码
5. 手动调整样式（可选）
6. **File** → **Export as** → **PNG**

## 方法三：ProcessOn

1. 访问：https://www.processon.com/
2. 新建流程图
3. 参考Mermaid代码，使用图形库手动绘制
4. ProcessOn支持更丰富的样式和图标

## 方法四：VS Code

1. 安装插件：**Markdown Preview Mermaid Support**
2. 打开 `.mmd` 文件
3. 右键 → **Markdown: Open Preview**
4. 截图保存

## 快速复制链接

## 常见问题

**Q: 中文显示乱码？**
A: 使用在线编辑器通常不会有此问题。VS Code需要安装支持中文的字体。

**Q: 图表太大？**
A: 在Mermaid代码中调整 `style` 的 `stroke-width`，或在导出时缩放。

**Q: 颜色不满意？**
A: 修改代码中的 `fill` 和 `stroke` 颜色值。
