# 使用手册

## 1. 首次配置

### 1.1 登录系统
- 访问系统地址，输入用户名和密码登录
- 默认管理员：`admin / 123456`（请立即修改）

### 1.2 配置大模型
1. 进入「模型」页面
2. 点击「添加模型」
3. 填写模型信息：
   - **Provider**：选择大模型厂商（推荐 DeepSeek 或智谱清言）
   - **API Key**：从对应平台获取
   - **Model Name**：填写模型名称（如 deepseek-chat）
   - **Base URL**：填写 API 地址
4. 点击「测试」确认可用
5. 点击「激活」设为默认模型

### 1.3 配置课程信息
1. 进入「课程」页面
2. 点击「新增课程」
3. 填写课程名称、专业、学段、教学模式、学生情况、默认班级
4. 点击「激活」设为当前课程

## 2. 教案生成

### 2.1 单章节模式
适用于生成单个章节的教案。

1. 进入「教案」页面
2. 选择「单章节」模式
3. 填写章节标题（如「第一章 Java 开发入门」）
4. 填写周次（如「2」）
5. 上传 PPT/PDF 素材（至少一个）
6. 可选：上传教学计划文件
7. 选择内容等级
8. 点击「生成」

### 2.2 按文件配置模式（推荐）
适用于多章节批量生成，每个文件独立配置。

1. 选择「按文件配置」模式
2. 上传多个 PPT/PDF 文件
3. 为每个文件配置：
   - 章节标题
   - 起止周次
   - 教案数量（即课时数）
4. 系统会自动做知识点规划，确保多份教案不重复
5. 点击「生成」

### 2.3 内容等级说明

| 等级 | 适用场景 | 教学过程段数 | 特色 |
|------|---------|------------|------|
| 基础版 | 快速备课 | 4 段 | 简明扼要 |
| 标准版 | 日常教学 | 6 段 | 详略适中 |
| 详尽版 | 公开课/示范课 | 8 段 | 含分层任务 |
| 特详版 | 教学检查/送审 | 10 段 | 含评价量规+评价细则 |

### 2.4 生成后操作
- 在「任务中心」查看实时进度
- 完成后可在「历史记录」中下载 Word 文档
- 支持预览教案内容
- 支持重新生成（重跑）

## 3. 题库生成

1. 进入「题库」页面
2. 上传章节素材文件
3. 选择模式（单章节或按文件配置）
4. 设置题型数量：
   - 单选题：推荐每章 5-10 道
   - 多选题：推荐每章 2-4 道
   - 判断题：推荐每章 3-5 道
   - 编程题：推荐每章 1-2 道
5. 设置难度倾向（简单/一般/困难/混合）
6. 选择内容等级
7. 点击「生成」

## 4. 知识库管理

### 4.1 上传资料
1. 进入「知识库」页面（在「更多」菜单中）
2. 点击「上传资料」
3. 填写标题，选择文件（支持 PDF/Word/PPT/TXT）
4. 系统自动解析并建立检索索引

### 4.2 知识库作用
- 生成教案/题库时，系统会自动从知识库检索补充内容
- 提高生成内容的专业性和准确性

## 5. 智能答疑

1. 进入「智能答疑」页面（在「更多」菜单中）
2. 在输入框中输入问题
3. 系统从知识库检索相关内容，结合 AI 给出回答
4. 支持追问和连续对话

## 6. AI 审阅

可对已生成的教案或题库进行 AI 专业评审：
- 评审教学目标的明确性和可衡量性
- 检查重难点是否准确
- 评估教学过程的完整性
- 提供具体改进建议

## 7. 管理员功能

### 7.1 用户管理
- 新增/编辑教师账号
- 启用/禁用用户
- 重置密码

### 7.2 仪表盘
- 查看系统使用情况统计
- 查看学情分析数据
- 查看任务趋势和模型使用分布

### 7.3 系统配置
- 配置生成参数（如自检开关、编译校验开关等）
- 管理邀请码
- 查看审计日志

## 8. 提升生成质量的关键设置

1. **模型选择**：推荐 qwen-max / glm-4-plus / deepseek-chat
2. **素材质量**：上传完整、可读的 PPT/PDF（建议 > 10 页）
3. **章节切分**：使用按文件配置模式，每章一个文件
4. **教案数量**：4学时/周 → 2份教案；2学时/周 → 1份教案
5. **题型分布**：每章至少含 5 单选 + 2 多选 + 3 判断
6. **失败重试**：任务失败积分自动退回，可在任务中心重跑
